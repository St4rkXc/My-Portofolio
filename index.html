My Skill Set

Figma

Figma adalah alat desain UI/UX yang saya gunakan untuk membuat prototipe dan desain interaktif. Pengalaman saya dengan Figma meliputi pembuatan wireframes, desain visual, dan prototipe yang mendetail.

HTML

HTML adalah bahasa dasar yang saya gunakan untuk membangun struktur dan kerangka halaman web. Saya memiliki pemahaman mendalam tentang HTML5, termasuk penggunaan tag semantik, pengelolaan elemen-elemen form, serta optimasi untuk SEO.

CSS

CSS adalah alat utama saya dalam menciptakan tampilan dan nuansa yang menarik untuk halaman web. Saya mahir dalam menggunakan CSS3 untuk mendesain layout yang responsif dan estetis, menguasai teknik seperti Flexbox dan Grid untuk tata letak, serta animasi untuk meningkatkan pengalaman pengguna.

Robotika

Robotika menggabungkan pengetahuan saya tentang perangkat keras dan pemrograman untuk menciptakan sistem otomatis yang kompleks. Saya memiliki pengalaman dalam merancang, membangun, dan memprogram robot menggunakan berbagai mikrokontroler, sensor, dan aktuator.
